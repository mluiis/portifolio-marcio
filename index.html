HABILIDADES

Ferramentas que possuo conhecimento buscando sempre a melhora.

HTML-5

O HTML é utilizado para estruturar o conteúdo na web. Essencial para o desenvolvimento de páginas. Sigo em constante evolução para dominar essa ferramenta.

CSS-3

O CSS tem a tarefa de estilizar as páginas. Onde você define uma identidade para seu site. Sigo em constante evolução para dominar essa ferramenta.

JAVASCRIPT

Linguagem de programação utilizada principalmente em páginas web. Permite criar interfaces dinâmicas, fazer verificações conforme sua aplicação.

BOOTSTRAP

Um dos framework mais utilizados no front-end, fornece estruturas CSS pré definidas para criação de sites e aplicações responsivas, para PC ou dispositivos móveis.

DIALOGFLOW

Ferramenta da Google, permite construções de chatbots para aplicação em aplicativos e dispositivos atráves de conversação baseada em voz e texto.

GITHUB

Maior repositório de dados compartilhados do mundo. Onde é possivel divulgar seus trabalhos e permitir que outros membros façam contribuições.
